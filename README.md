# Custom Audio Player (HTML, CSS, JavaScript)
  Project Overview

This project is a custom audio control interface made with just HTML, CSS, and JavaScript.
Without using any outside libraries, it lets users interact with an audio file through features that are controlled by code.

The main goal of this project is to show how JavaScript can be used to make web apps that are interactive and control media elements.

Characteristics
 Audio Play and Pause
Users can use buttons to start and stop audio playback.
 Control the Volume
Users can change the volume up or down with a slider.
 Go Forward or Back
Users can move the audio forward or backward by 10 seconds.
 Play Automatically After End
When the audio is done, it plays again on its own.
 Display of Dynamic Time
Shows the current playback time and the total length of the video.

Technologies Used
HTML5
CSS3
JavaScript
